borea.datastruct.dtm module¶

A module for manipulating a digital elevation model.

class borea.datastruct.dtm.Dtm(*args, **kwargs)[source]¶

Bases: WorldImageDtm

Represents a digital elevation model.

Parameters:

  • path_dtm (str) – Path to the dtm file. If None Dtm.get() always returns 0.

  • order (int) – The method of interpolation to perform. 0 : nearest, 1 : slinear, 3 : cubic, 5 : quintic.

  • cval (int) – Value to fill past edges of dtm. If None raise an error for any point outside the dtm.

  • keep_in_memory (bool) – Store all data in memory.

Note

All gdal formats are supported. The file must contain georeferencing information.
